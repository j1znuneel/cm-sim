# CM SIM: KERALA

**CM SIM: KERALA** is a retro-style fiscal simulator game where you step into the shoes of the Chief Minister of Kerala. Your mission: navigate the state through a decade of fiscal turbulence from 2026 to 2036.

![CM SIM: KERALA Screenshot](https://via.placeholder.com/800x400.png?text=CM+SIM:+KERALA+Gameplay)

## 🎮 The Mission

Balance the "Kerala Model" until 2036. You must manage the state's finances, public trust, and debt while enacting social welfare programs and infrastructure projects.

### Core Metrics:
*   **💰 CASH (Liquid Reserves):** Must stay above 0. If reserves empty, trust decays rapidly.
*   **📉 OMB DEBT:** Limited to an additional ₹45,000 Cr over the starting debt. Exceeding this leads to federal takeover.
*   **💙 TRUST (Public Trust):** You need at least 67% trust by the 2031 election to secure a second term. Falling to 0% leads to resignation.

## 🚀 Key Features

*   **Retro CRT Aesthetic:** Immerse yourself in a pixel-perfect interface with CRT scanline effects and a VT323 monospace font.
*   **Fiscal Management:** Adjust quarterly allocations for Healthcare, Education, KIIFB, Agriculture, Welfare, and IT/Industry.
*   **Manifesto Items:** Enact major policies like Free KSRTC Bus, Student Stipends, and Welfare Pension hikes to boost trust, but beware of the permanent quarterly burn.
*   **Treasury Actions:** Issue OMB Loans (Open Market Borrowing) for instant cash at the cost of long-term interest and debt ceiling pressure.
*   **Dynamic News Ticker:** Stay updated with real-time news alerts on GST disputes, infra projects, and election warnings.

## 🛠️ Tech Stack

*   **Frontend:** HTML5, Vanilla JavaScript
*   **Styling:** Tailwind CSS (via CDN), Custom CSS for CRT effects
*   **Typography:** Google Fonts (VT323)
*   **Assets:** CSS-based pixel art avatars

## 🕹️ How to Play

1.  **Allocate Resources:** Use the sliders to distribute funds across different sectors every quarter. Spending more than ₹3,500 Cr per sector helps grow trust.
2.  **Enact Policies:** Use your reserves to fulfill manifesto promises for large trust gains.
3.  **Manage Debt:** Use OMB Loans sparingly when cash is low.
4.  **Survive to 2036:** Secure your legacy by balancing the impossible.

## 🛠️ Development

This is a single-file application (`index.html`). To run it locally:

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
